## Start one

```bash
pm2 start lessons/10-nodejs-useful-npm-modules/examples/pm2/server.js
```

## Start 4 instances

```bash
pm2 start lessons/10-nodejs-useful-npm-modules/examples/pm2/server.js -i 4
```

pm2 monit
